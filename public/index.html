The Board Game Awards Database

A RESTful web service to obtain board game award information, all content contributed and maintained by the gaming community.

üìã Usage

Send all data requests to:

https://gameawards.netlify.app/api/?apikey=[yourkey]&

Parameters

  • i - Award ID (e.g. 10865)
  • t - Award title to search for
  • s - Search across all fields
  • bgg_id - BoardGameGeek game ID
  • year - Year of award
  • category - Award category/position
  • award_set - Award set name

üîç Examples

Get award by ID:
?i=10865&apikey=yourkey
Search for "Spiel des Jahres":
?s=Spiel des Jahres&apikey=yourkey
Get all awards for a game:
?bgg_id=361&apikey=yourkey
Awards from 2020:
?s=winner&year=2020&apikey=yourkey
